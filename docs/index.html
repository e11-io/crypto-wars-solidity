Crypto Wars

This repository holds all the smart contracts that are going to be used on the game Crypto Wars. Please check the documentation for more details.

Installation

  1. Install truffle and an ethereum client. For local development, try EthereumJS TestRPC.

    npm install -g truffle
npm install -g ethereumjs-testrpc
truffle version
# Truffle v4.0.1 (core: 4.0.1)
# Solidity v0.4.18 (solc-js)

  2. Install dependencies.

    npm i # or yarn

  3. Run testrpc on a separate terminal

    npm run rpc

  4. Compile the contracts.

    truffle compile

  5. Run the tests.

    truffle test

  6. Compile the docs.

    npm run doxity init
npm run docs

  7. Run test coverage.

    npm run coverage

  8. Migrate the contracts.

    truffle migrate

  9. Run the web app.

    npm run start # and navigate to http://localhost:4200
